Installation¶

polar-bites is hosted on github, and can probably be installed using your favourite package manager.

poetry¶

Add a requirement to the github link to the [tool.poetry.dependencies] section in your pyproject.toml file.

[tool.poetry.dependencies]
polar-bites = {git = "https://github.com/maxastyler/polar-bites.git"}

You can control which version you use by commit, or tag. See the poetry docs for that.

Once you’ve put the dependency in your pyproject.toml file, update the poetry virtual environment:

poetry update
