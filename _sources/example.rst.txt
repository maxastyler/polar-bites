Examples
========

A list of things you might want to do with :doc:`polar-bites <index>`.

Loading MATLAB files
--------------------

The function :func:`polar_bites.mat.load_mat_to_dataframe` is used to extract a structure array from a given ``.m`` file. 

The Column
``````````
The :class:`Column<polar_bites.column.Column>` class helps to load columns from a MATLAB data file.
