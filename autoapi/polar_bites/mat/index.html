polar_bites.mat.load_mat_to_dict(filename, var_name=None, array_transform_keys=None)¶

Load a .mat file to a dictionary. var_name should be the variable that the data is stored under in the mat file. If this isn’t given, then the first variable is loaded. Nested cells are flattening by joining their names with underscores The keys in array_transform_keys are converted from numpy arrays to lists of floats

Parameters:

  • filename (Union[pathlib.Path, str]) – the .mat’s filename

  • var_name (Optional[str]) – the variable to extract from the mat file

  • array_transform_keys (Optional[Sequence[str]]) – the list of keys in the structure which should be converted from numpy arrays to lists of float

Return type:

dict[str, Any]
